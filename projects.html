My Top Projects

I completed a range of projects during the She Codes Plus program, in front end and back end development. Technologies used: Python; Javascript; Django; React; HTML; CSS; Javascript; Github; Heroku.

Technologies used: DRF | React | HTML | CSS

This project was our final group project for the She Codes Plus 2021 Perth cohort. We worked with a client that gave us requirements for a website that would display sticky notes from the different She Codes events. We had three weeks to work as a team to provide a solution that integrated DRF for the backend and React for the frontend and deployed on Heroku. Using QR codes attendees were able to post their feedback on a live WinWall that sat in it's own collection of WinWalls either anonymously or with a user account. Superusers were able to export the user and feedback data as a csv file.
